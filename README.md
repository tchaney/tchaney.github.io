# Thomas Chaney

Static GitHub Pages site for <https://tchaney.github.io>, rebuilt from the public Google Site at <https://sites.google.com/site/thomaschaney/home>.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static file server:

```sh
python3 -m http.server 8000
```

## Structure

- `index.html` - home and contact page
- `research.html` - working papers, publications, and archived drafts
- `cv.html` - web CV highlights and full PDF link
- `teaching.html` - lecture notes, slides, and course archive
- `misc.html` - legacy miscellaneous page
- `assets/` - shared stylesheet and visual assets
